1. Общее описание:
   Система предназначена для предоставления пользователям доступа к электронным книгам, аудиокнигам и управления личной цифровой библиотекой.

2. Роли пользователей (Акторы):
* Гость: Неавторизованный посетитель сайта.
* Читатель: Зарегистрированный пользователь, имеющий активную подписку.
* Автор: Зарегистрированный пользователь, который может публиковать свои произведения.
* Администратор: Управляет контентом, пользователями и системой.

3. Функциональные требования (сгруппированные по акторам):

* Для Гостя:
    - FR1.1: Просмотр каталога книг с базовой фильтрацией (по жанру, популярности).
    - FR1.2: Регистрация в системе путем создания учетной записи (email, пароль).
    - FR1.3: Просмотр подробной информации о конкретной книге (описание, автор, рейтинг, оглавление).

* Для Читателя (включает все возможности Гостя +):
    - FR2.1: Вход в систему (аутентификация).
    - FR2.2: Поиск книг по расширенным критериям (название, автор, ключевые слова, ISBN).
    - FR2.3: Добавление книги в "Мои книги" (личная коллекция).
    - FR2.4: Чтение/прослушивание книги онлайн из личной коллекции.
    - FR2.5: Создание и управление закладками в книге.
    - FR2.6: Оставление рецензии и оценки (от 1 до 5 звезд) для книги.
    - FR2.7: Управление профилем (смена пароля, редактирование аватара).
    - FR2.8: Оформление и управление подпиской (оплата, отмена, просмотр статуса).

* Для Автора (включает все возможности Читателя +):
    - FR3.1: Загрузка новой книги в систему (PDF, EPUB файлы) с метаданными.
    - FR3.2: Управление своими загруженными книгами (редактирование описания, обновление файла, снятие с публикации).
    - FR3.3: Просмотр статистики по своим книгам (количество чтений, средний рейтинг).

* Для Администратора:
    - FR4.1: Модерация загруженных книг (одобрение/отклонение публикации).
    - FR4.2: Управление пользователями (блокировка, назначение роли "Автор").
    - FR4.3: Управление жанрами и категориями каталога.
    - FR4.4: Просмотр системной аналитики (общая активность, финансовые отчеты по подпискам).

4. Нефункциональные требования:
* NFR1: Система должна обслуживать до 10 000 одновременных пользователей.
* NFR2: Время отклика на любой запрос не должно превышать 2 секунд.
* NFR3: Интерфейс для чтения книг должен корректно отображаться на устройствах с экранами от 4 до 27 дюймов.
